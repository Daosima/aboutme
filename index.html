What is HTML

Можете считать, что HTML и CSS - это базовые кирпичики из которых создаётся отображение веб-страницы в вашем браузерею
И делается это непосредственно руками верстальщика. Желательно, чтобы это происходило по заранее подготовленному макету дизайнера...

Layout

Fixed

Контент отображается одинаково на всех типах устройств и разрешениях экрана - фиксированно

Adaptive

Содержание страницы корректно подстраиваться под любое устройство, с которого открыт сайт

Cross-browser

Сайта одинакого отображается во всех популярных браузерах: Chrome, Firefox, Opere, IE, Safari и другие

Flex

Flexible Box Layout Module - блочная модель для гибкого управления контентом на сайте

Semantic

Подразумевает использование HTML-тегов в соответствии с их предназначением, что очень любят поисковые роботы

validation

Валидная - вёрстка без ошибок, в соответствии со стандартами W3C
